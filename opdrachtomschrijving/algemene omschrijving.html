Cijferopbouw:

Je uiteindelijke cijfer voor dit PO wordt voor een derde bepaald door de tussenopdrachten en voor twee derde door de eindopdracht die je inlevert.

Tussenopdrachten:

Voor de tussenopdrachten geldt een pass/fail-systeem. Per tussenopdracht verdien je 1,5 punt voor je uiteindelijke PO-cijfer, als je deze op tijd inlevert en deze inhoudelijk in orde is (pass). Bij een fail krijg je hier geen punten voor. Je haalt een fail op je tussenopdracht in de volgende gevallen:

  • Je levert de tussenopdracht niet op tijd in.
  • Je ingeleverde tussenopdracht voldoet duidelijk niet aan de eisen. (Zie hieronder.)
  • Je ingeleverde tussenopdracht is duidelijk met ChatGPT of andere genAI gemaakt.

Als je alle tussenopdrachten succesvol inlevert heb je dus al minstens een 4 (1 + 1,5 + 1,5 = 4) voor je PO. Als je geen enkele tussenopdracht succesvol inlevert, haal je maximaal een 7 (10 - 1,5 - 1,5 = 7).

Eindopdracht:

Eisen (1,5 punt te behalen):

  • Alle eisen waaraan de tussenopdrachten moeten voldoen. (Zie hieronder)
  • Je website maakt gebruik van een aparte stylesheet.
  • Je gebruikt selectors voor verschillende elements en classes.
  • Je gebruikt het boxmodel (ergens) in je website: je past borders, margins, padding en height of width aan.

Verdere beoordelingscriteria (4,5 punt te behalen):

  • Hoe uitgebreid is je site? Voldoe je enkel aan de basisvereisten of heb je zelf nog dingen toegevoegd?
  • HTML-/CSS-vaardigheid. (Pas je alleen toe wat in de les aan bod is gekomen of laat je zien dat je meer kan?)
  • Gebruikerservaring: Is de site mooi/overzichtelijk? Kan de gebruiker dingen aanpassen? Heeft alles een logische naam? Werken alle links?
  • Netheid: Staan alle bestanden op een logische plek? Gebruik je comments in je code? Is je code overzichtelijk?
